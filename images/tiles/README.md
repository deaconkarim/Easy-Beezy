# Tile Images

Place your unique images for each tile value in this directory.

## Required Images

The game uses powers of 2 for tile values. You need to create images named:

- `2.png` - for tile value 2
- `4.png` - for tile value 4
- `8.png` - for tile value 8
- `16.png` - for tile value 16
- `32.png` - for tile value 32
- `64.png` - for tile value 64
- `128.png` - for tile value 128
- `256.png` - for tile value 256
- `512.png` - for tile value 512
- `1024.png` - for tile value 1024
- `2048.png` - for tile value 2048
- `4096.png` - for tile value 4096 (Beezy's Babies goal!)
- `8192.png` and higher for super tiles (optional)

## Image Recommendations

- Square images work best (e.g., 200x200px or 256x256px)
- PNG format with transparency is recommended
- Images will be scaled to fit within tiles
- Keep file sizes reasonable for fast loading
